# x86 Assembly Examples

A collection of small MASM x86 assembly programs created for learning and practicing low-level programming concepts.

These programs demonstrate:
- Procedures and stack frames
- Arrays and pointer arithmetic
- Loops and conditional branching
- Bitwise operations
- String manipulation
- Structures
- Memory addressing
- Register-based arithmetic

## Programs

### 0. Array Summation
Calculates the sum of an integer array using pointer arithmetic and loops.

### 1. Even or Odd Checker
Determines whether an integer is even or odd using bitwise operations.

### 2. Fibonacci
Generates and stores Fibonacci numbers in memory.

### 3. Extract Date
Extracts day, month, and year fields from a packed 16-bit value.

### 4. Largest Array Element Finder
Traverses a signed integer array to locate the largest value.

### 5. PIN Range Validator
Validates PIN digits against minimum and maximum allowed ranges.

### 6. Student Structure Processing (Decimal Values)
Performs arithmetic operations on structured student data.

### 7. Student Structure Processing (Hex Values)
Uses structures to store student records and calculate quiz totals.

### 8. Substring Search
Searches for a substring inside a target string using string comparison instructions.

### 9. Sum and Average Calculator
Computes the sum and average of three integers using stack-based procedure parameters.

### 10. Uppercase String Converter
Converts lowercase ASCII characters in a string to uppercase.

## Technologies

- x86 Assembly Language
- MASM (Microsoft Macro Assembler)
